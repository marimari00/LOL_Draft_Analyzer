## Accessing the Application

After starting:
- **Frontend:** Browser opens automatically to `http://localhost:3000`
- **Backend API:** Running at `http://localhost:8000`

---

## Stopping the Servers

Close the terminal windows that opened, or press `Ctrl+C` in each window.

---

## Troubleshooting

**Problem:** "npm not recognized"  
**Solution:** Close all PowerShell/CMD windows and try again. Node.js needs a fresh terminal.

**Problem:** Port already in use  
**Solution:** Close any existing backend/frontend terminals first.
